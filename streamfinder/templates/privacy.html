Privacy Policy

The only resource I've really put into this site is my time, and so I'm not going to pay an expensive lawyer to write something here that only other lawyers can really understand. So here's the rundown from me:

  • The server records every request and its originating IP address in a log file. This is helpful when fixing bugs, and for determining which parts and features of this site are actually useful and which aren't.
  • This site uses an authentication cookie to determine whether you're logged in or not, and which account you're logged in under. It is cryptographically signed to prevent tampering.
  • When logging in via Steam or Twitch, only the most basic permissions are requested from you in order to authenticate. After logging in, your unique Steam identifier or Twitch identifier is stored in the database, along with your account name and picture URL. All of these stored values are accessible through the public APIs of Steam or Twitch without any authentication.
